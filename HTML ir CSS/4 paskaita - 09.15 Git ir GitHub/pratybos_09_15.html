Naudojant bet kurią apart Static pozicijos, gauname prieigą prie 5 naujų stilių: top, left, right, bottom, z-index.
Top/left/right/bottom stiliai nurodo per kokį atstumą elementas turi būti atitrauktas.
Z-index stilius nurodo elemento prioritetą sluoksniuose (kuris elementas užlips ant kurio elemento (esant persidengimui kuris elementas bus viršuje/apačioje)).

Static pozicija - nieko ypatingo nedaro. Net nesuteikia galimybės naudoti top/left/right/bottom/z-index;

Terminalo komandos

  • cd - change directory - pakeisti direktoriją - vaikščiojimas tarp aplankalų
  • ls - list files and directories - išvardinti failus ir aplankalus
  • CTRL C - atšaukia vykdomą veiksmą
  • TAB - užbaigia frazę arba pateikia pasiūlymus
  • clear - nukelia visą terminale rodomą informaciją į viršų

Git komandos

  • init - initialize - inicializuoti - sukuria git repozitoriją
    • status - statusas - pateikia informaciją apie tavo faiilus esamoje repozitorijoje
  • add - stage'ina/prideda/paruošia - paruošia failus comit'inimui
  • commit - sukuria kodo versiją iš stage'intų failų
  • push - įstumti - lokalius (jau su'comit'intus) pakitimus perduoda išorinei aplinkai (Github'ui)
  • pull - traukti - GitHub'e esančius pakitimus atsisiunti pas save lokaliai
